# Currency Converter

A simple currency converter that allows users to convert between different currencies using live exchange rates fetched from a public API.

## Technologies Used:
- HTML, CSS, JavaScript
- Fetch API
- Async/Await, Promises, Callbacks

## Features:
- Real-time currency exchange rate fetching
- Conversion between multiple currencies
- Responsive design for mobile devices

## How to Run:
1. Clone this repository
2. Open the index.html file in your browser
3. Enjoy converting currencies!
